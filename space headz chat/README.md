# Space Headz chat

Application written from scratch, enabling real time communication over different clients - Chrome, Edge, Firefox...

### - App Stack -

##### Back End:
- Node.js Express + WebSocket module
##### Front End: 
- HTML5/CSS3 + client WebSocket & DOM manipulation.
